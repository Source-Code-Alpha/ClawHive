# Skill: Brainstorming

Facilitate structured ideation sessions that produce actionable ideas.

## Methodology: Diverge then Converge

### Phase 1: Diverge (Generate)
- **No judgment** -- every idea is valid during this phase
- **Quantity over quality** -- aim for 15-20 ideas minimum
- **Build on others** -- "yes, and..." not "but..."
- **Go wild** -- unrealistic ideas often inspire realistic ones

### Phase 2: Converge (Evaluate)
- **Group similar ideas** into themes
- **Score each theme** on: Impact (1-5), Feasibility (1-5), Effort (1-5)
- **Prioritize** using Impact x Feasibility / Effort
- **Select top 3** for deeper exploration

### Phase 3: Develop (Refine)
For each top idea:
- One-paragraph description
- Key assumption to validate
- First concrete next step
- Potential blocker

## Output Format

```markdown
## Brainstorm: [Topic]

### All Ideas (diverge)
1. ...
2. ...

### Themes
- Theme A: ideas 1, 3, 7
- Theme B: ideas 2, 5, 8

### Top 3 (converge)
1. [Idea] -- Impact: X, Feasibility: Y
2. ...
3. ...

### Next Steps
- ...
```
